# 🚀 AMEVA Universal CORS Unblocker Extension

이 확장 프로그램은 **Chrome, Edge, Naver Whale, Opera, Firefox** 등 주요 웹 브라우저에서 AMEVA Web Downloader가 타사(유튜브, 틱톡, 인스타) 서버와 통신할 때 **CORS 보안 차단을 우회**할 수 있도록 도와주는 범용 확장 프로그램입니다.

---

## 🌐 브라우저별 설치 방법 (1분 소요)

### 1. Google Chrome (구글 크롬)
1. 크롬 주소창에 `chrome://extensions` 입력 후 엔터.
2. 우측 상단의 **'개발자 모드 (Developer mode)'** 스위치를 **켜기**.
3. 좌측 상단 **'압축해제된 확장 프로그램을 로드합니다 (Load unpacked)'** 버튼 클릭.
4. 이 `extension` 폴더를 선택하면 끝!

---

### 2. Microsoft Edge (마이크로소프트 엣지)
1. 엣지 주소창에 `edge://extensions` 입력 후 엔터.
2. 좌측 메뉴 하단의 **'개발자 모드'** 스위치를 **켜기**.
3. **'압축 해제된 로드'** 버튼 클릭 후 이 `extension` 폴더 선택.

---

### 3. Naver Whale (네이버 웨일)
1. 웨일 주소창에 `whale://extensions` 입력 후 엔터.
2. 우측 상단의 **'개발자 모드'** 스위치를 **켜기**.
3. **'압축 해제된 확장 앱 로드'** 버튼 클릭 후 이 `extension` 폴더 선택.

---

### 4. Opera (오페라)
1. 오페라 주소창에 `opera://extensions` 입력 후 엔터.
2. 우측 상단의 **'Developer mode'** 스위치를 **켜기**.
3. **'Load unpacked'** 버튼 클릭 후 이 `extension` 폴더 선택.

---

### 5. Mozilla Firefox (파이어폭스)
1. 파이어폭스 주소창에 `about:debugging#/runtime/this-firefox` 입력 후 엔터.
2. **'임시 부가 기능 로드 (Load Temporary Add-on...)'** 버튼 클릭.
3. `manifest.json` 파일을 직접 선택하면 즉시 활성화됩니다.

---

## ✅ 설치 후 확인
확장 프로그램을 켜두신 상태에서 [AMEVA Web Downloader](https://uno-km.github.io/scrapy_bot/web_downloader/) 웹사이트에 접속하시면, CORS 차단 없이 영상 및 사진 추출이 원활하게 진행됩니다!

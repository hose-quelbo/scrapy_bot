🔌 확장 프로그램 가이드

📦 확장 프로그램 다운로드

Google Chrome (구글 크롬) 설치 방법:

  1. 상단의 📦 확장 프로그램 다운로드 버튼을 눌러 압축파일을 받고 해제합니다.
  2. 크롬 주소창에 chrome://extensions 입력 후 엔터. (클릭 시 복사됨)
  3. 우측 상단의 '개발자 모드' 스위치를 켭니다.
  4. 좌측 상단 '압축해제된 확장 프로그램을 로드합니다' 버튼을 누르고 압축 해제된 폴더 선택!

Microsoft Edge (마이크로소프트 엣지) 설치 방법:

  1. 엣지 주소창에 edge://extensions 입력 후 엔터. (클릭 시 복사됨)
  2. 좌측 메뉴 하단의 '개발자 모드' 스위치를 켭니다.
  3. '압축 해제된 로드' 버튼 클릭 후 다운받은 폴더 선택.

Naver Whale (네이버 웨일) 설치 방법:

  1. 웨일 주소창에 whale://extensions 입력 후 엔터. (클릭 시 복사됨)
  2. 우측 상단의 '개발자 모드' 스위치를 켭니다.
  3. '압축 해제된 확장 앱 로드' 버튼 클릭 후 다운받은 폴더 선택.

Opera (오페라) 설치 방법:

  1. 오페라 주소창에 opera://extensions 입력 후 엔터. (클릭 시 복사됨)
  2. 우측 상단의 'Developer mode' 스위치를 켭니다.
  3. 'Load unpacked' 버튼 클릭 후 다운받은 폴더 선택.

Mozilla Firefox (파이어폭스) 설치 방법:

  1. 파이어폭스 주소창에 about:debugging#/runtime/this-firefox 입력 후 엔터. (클릭 시 복사됨)
  2. '임시 부가 기능 로드 (Load Temporary Add-on...)' 버튼 클릭.
  3. 압축 푼 폴더 내부의 manifest.json 파일 선택.
